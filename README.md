# Бот проверки неучтённого товара

Telegram-бот на **Aiogram 3** и **SQLite** для ежедневной отработки неучтённых артикулов по отделам (1–15).

## Возможности

- Загрузка Excel в любое время (формат `Неучтенные товары.xlsx`)
- **10 артикулов в день** на отдел, без повторов до следующей загрузки
- Кнопки **«Отработан»** / **«Не отработан»** (без фото)
- При входе в отдел: *«Сегодня ДД.ММ.ГГГГ, нужно отработать эти артикулы»* + карточки
- Планировщик: обновление артикулов и **сводка по отделам** (кто отработан / ещё нет) — часы настраиваются
- Админ-кнопка **⚙️ Настройки**: время обновления, часы сводки, артикулов в день
- Когда все артикулы дня отмечены «Отработан» — сообщение *«Все артикула на сегодня отработаны…»*

## Настройка (.env)

```env
BOT_TOKEN=
ADMIN_USER_IDS=123456789
PRODUCTS_PER_DAY=10
DAILY_ASSIGN_HOUR=6
SUMMARY_HOURS=10,12,14,16
TIMEZONE=Europe/Moscow
```

## Запуск

```bash
pip install -e .
python -m store_check_bot.main
```

## Excel

Колонки как в образце: Магазин, Отдел, Подотдел, Артикул, Наименование, LOC, Гамма, ТОП, Теор. запас, …

В БД сохраняется поле **shown_for_check_date** — дата вывода артикула на проверку.
